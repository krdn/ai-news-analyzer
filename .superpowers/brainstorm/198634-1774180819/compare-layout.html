셀럽 비교 페이지 — 시각화 구성

2~4명의 셀럽을 나란히 비교할 때 어떤 조합이 좋을까요?

A

감성 추이 오버레이 + 요약 카드

하나의 라인 차트에 셀럽별 감성 추이를 색상으로 구분. 상단에 각 셀럽의 현재 감성 점수/댓글 수 요약 카드.

비교 대시보드
셀럽 A
+0.45
댓글 1,234개
셀럽 B
-0.32
댓글 856개
— 셀럽 A    — 셀럽 B

장점

  • 한눈에 추이 비교
  • 직관적

단점

  • 주제별 비교 불가
B

오버레이 차트 + 레이더 차트 (주제별)

A에 더해, 주제별 감성을 레이더 차트로 비교 (연기력, 인성, 외모 등). Phase 2B의 topics 데이터 활용.

비교 대시보드 + 레이더
주제별 비교

장점

  • 추이 + 주제별 비교 모두 가능
  • Phase 2B 데이터 최대 활용

단점

  • 레이더 차트 구현 추가 필요
C

오버레이 + 레이더 + 소스별 비교 테이블

B에 더해, 소스별(네이버/유튜브/X 등) 감성 비교 테이블. 어떤 플랫폼에서 어떤 셀럽이 더 인기인지.

소스별 비교
소스 셀럽 A 셀럽 B
네이버 +0.52 -0.21
유튜브 +0.38 -0.45
X +0.61 +0.02

장점

  • 가장 풍부한 비교
  • 플랫폼별 여론 차이 파악

단점

  • 구현 범위 가장 큼
